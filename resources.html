Pathogen Genomics

PHA4GE

The activities within PHA4GE are aligned to its vision of achieving a rapid global genomic-driven public health response to disease outbreaks. Building the data standards, and sharing best practices require the input from global stakeholders. PHA4GE has defined eight working groups with deliverables determined by public health needs. These include:

  1. Data structures
  2. Infrastructure
  3. Bioinformatics pipelines and visualization
  4. Public sequence repositories
  5. Reference qulaity control and validation
  6. Training and workforce development
  7. Users and applications
  8. Ethics and data sharing
Each of these working groups benefit from collaborative work among its members https://pha4ge.org/.

IDseq

IDseq is user-friendly software platform, originally developed for metagenomics studies. Functionality has since been added for SC2 consensus assembly calling from paired-end Illumina data.
https://www.discoveridseq.com/

Basestack

Basestack is a platform for local deployment of standardized genomic surveillance analysis tools. It currently enables real-time analysis of Nanopore generated sequencing data for SARS-CoV-2.
https://jhuapl-bio.github.io/Basestack/
